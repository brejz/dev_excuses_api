Developer Excuses API

In this hectic world, developers are often faced with the problem of having to come up with excuses on the spot to satitisfy clients or their employers. Developers should no longer have to worry about such mundane tasks but use the Developer Excuses API for all their excuses requirements.

Example Calls:

  • All /api/v1/resources/excuses/all
  • Id specific /api/v1/resources/excuses?id=1
  • Type specific calls ('Jobs or Client') /api/v1/resources/excuses?type=Job
